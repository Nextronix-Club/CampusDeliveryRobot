# CampusDeliveryRobot

## Project Overview
Develop a campus delivery robot system using ESP32 CAM for autonomous delivery services within the college campus.

Each team member will provide their own perspective and approach to solve this campus delivery robot project. Create your `SOLUTION.md` file in your respective folder explaining how you would approach building this system.

## Project Requirements
- Campus delivery robot with ESP32 CAM
- Web/App interface for ordering
- Navigation system (left, right, forward, back commands)
- Error handling and obstacle avoidance
- Complete system architecture from your perspective
