# CampusDeliveryRobot

## Project Overview
Develop a campus delivery robot system using ESP32 CAM for autonomous delivery services within the college campus.

- Navigation system (left, right, forward, back commands)
- Error handling and obstacle avoidance
- Complete system architecture from your perspective
